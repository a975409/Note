## 取得 Domain Host 的方法

```C#
// 取得完整主機名稱（含協議、主機名、埠號）
var host = $"{Request.Scheme}://{Request.Host.Value}";

// 或只取得 Host（不含協議）
var hostOnly = Request.Host.Host;

// 取得埠號（如果有的話）
var port = Request.Host.Port;

```

- `Request.Scheme`：取得請求使用的協議（如 `http` 或 `https`）。
- `Request.Host`：包含主機名稱與埠號（類型為 `HostString`）。
- `Request.Host.Value`：主機名稱與埠號的字串表示（例如 `example.com:5001`）。
- `Request.Host.Host`：純主機名稱（不含埠號）

### 常用指令(git bash)

1. 查看當前目錄：
```bash
pwd
```

2. 創建目錄：
```shell
mkdir [路徑或目錄名稱]
```

3. 切換目錄：
```shell
cd [路徑或目錄名稱]
```

4. 清除 Terminal 畫面：
```shell
clear
```

5. 進入 claude功能：
```shell
claude
```
